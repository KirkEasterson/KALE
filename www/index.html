Installation

KALE can be installed on any derivative of Arch Linux that uses the Arch repositories (including Manjaro, Antergos and Parabola, etc.). I recommend you to install it on a fresh installation to avoid conflict with other desktop environments, although you can run it on an already existing install. Note: KALE will install config files in the home directory of the user you give it. It will not wipe your Documents or anything, but it will overwrite your ~/.bashrc, etc. If you have config files you want to hold on to, either back them up or you can select to install KALE to another user.

To install, simply log in as the root user and run the following in the terminal:

curl -LO kirkeasterson.com/kale.sh
bash kale.sh

This will begin a prompt and the installation after some setup. Installation time will vary depending on your internet connection. Don't be worried if it takes even a little more than an hour: it's setting everything up.
